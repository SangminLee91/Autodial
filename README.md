한빛 원자력발전소 발신번호 변경 시스템(개인 프로젝트)

[담당 업무]

로그인 / 계정 CRUD / 전화번호 CRUD / 교환기(전화기) 연동하여 사용중인 전화기 모니터링 / 일별, 월별 사용통계 / 일별, 월별 상세이력 / 통계 및 이력 엑셀 출력

언어, 프레임워크 및 라이브러리:

Front - Thymeleaf, jQuery

Back - Java, JavaScript, SpringBoot, MyBatis
